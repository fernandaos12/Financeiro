### Descrição:
App web para organizar as finanças pessoais integrando pagamentos e contas a pagar.

### Tecnologias:
1. .Net Core 9
2. API Rest
3. Angular
4. SqlServer
5. DockerFile

### Instalação e pre-requisitos
Docker
Visual Studio 2022 

### Instrução de uso
Utilizar o dockerfile para gerar imagem container .net core 9 e SqlServer
